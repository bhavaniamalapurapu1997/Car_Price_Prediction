# Car_Price_Prediction
Developed a predictive model to estimate used car prices based on features such as make, model, year, engine aspiration, drivetrain and transmission.
•	Performed data cleaning, feature engineering, and exploratory data analysis (EDA) to identify key price-influencing factors.
•	Implemented and analysed the accuracy, RMSE(root mean square error), MAE(mean absolute error) using Linear Regression.
•	Built an interactive Streamlit web app for real-time price prediction, enabling users to input car details and instantly view estimated prices.
